## jane
![cover](./cover.jpg)
+ scss、less、async、await、api的promise化处理

## Install
```
sudo npm install -g @linhun/jane
```

## Features
+ .scss -> .wxss 
+ es6 -> es5
+ async、await

## Quickstart
init project
``` sh
jane new project
```
build project
``` sh
jane build // on production
jane build -w // watch build on develop
```

## More usage
```
jane --help
```
